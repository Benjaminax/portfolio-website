\section{Project Summary}
This document describes the development of a third person shooter game built in Unreal Engine 5.7. The game uses a custom player character (\textbf{QuantumCharacter}), a stylized environment (\textbf{Stylized\_Egypt}), animation systems based on Unreal Animation Blueprints and Mixamo assets, and combat systems including line tracing, muzzle flash effects, and AI enemy damage.

\section{Implemented Features and Technical Work}
This section documents what was implemented during production.

\subsection{Animation and Character Systems}
\begin{itemize}[leftmargin=*]
  \item Created an Animation Blueprint for \textbf{QuantumCharacter}.
  \item Implemented sprint action input and movement-speed-driven transitions.
  \item Implemented crouch input and crouch animation behavior.
  \item Added weapon integration with animation-ready combat poses.
  \item Built an Animation State Machine for idle, walk, run, sprint, crouch, and combat states.
  \item Added aim offset animation for directional aiming support.
  \item Implemented switching between animation states based on gameplay inputs and velocity.
\end{itemize}

\subsection{Camera and Aiming Systems}
\begin{itemize}[leftmargin=*]
  \item Implemented camera zoom effect while aiming (ADS behavior).
  \item Linked aiming state to animation logic and combat flow.
  \item Fixed crosshair behavior to align aiming and hit registration expectations.
\end{itemize}

\subsection{Combat and Weapon Systems}
\begin{itemize}[leftmargin=*]
  \item Implemented line tracing from weapon muzzle direction for hit detection.
  \item Added muzzle flash visual effect and weapon shot audio.
  \item Added impact shot sound/effect feedback on successful hits.
\end{itemize}

\subsection{Enemy AI and Health Systems}
\begin{itemize}[leftmargin=*]
  \item Implemented enemy AI behavior and applied damage logic.
  \item Added AI Pawn Sensing for player detection/perception.
  \item Added health bars for both player and enemy.
\end{itemize}
